Network Basics

Below are all the detailed points you need to know about networking fundamentals:

Key Concepts

  • IP Address: Unique identity for each device
  • Subnet Mask: Divides network and host parts
  • Router: Connects networks
  • Switch: Connects devices within a network
  • Bandwidth: Max data transfer rate
  • Protocol: Rules for communication (TCP/IP, HTTP, HTTPS)

Types of Networks

  • LAN: Local Area Network
  • WAN: Wide Area Network
  • MAN: Metropolitan Area Network
